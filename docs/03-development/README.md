- **前端**: Vue 3 + Vite
- **后端**: Node.js + Express + MongoDB + TypeScript
- **工作流**: n8n自动化工作流引擎
- **AI集成**: Claude、Perplexity、ChatGPT等API
- **区块链**: Solana（主链），计划扩展SUI和TON
