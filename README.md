# EZ Trading - 游戏风格AI交易神谕终端

![EZ Trading Banner](./assets/banner.png)

## 📋 项目概述

**EZ Trading**（原Oracle Alpha）是一个面向加密货币散户的游戏化AI交易神谕平台，将复杂的交易体验转化为沉浸式科技仪式。平台通过精心设计的游戏化机制，让用户能够"像刷抖音一样轻松做交易"，同时体验代入感十足的高科技交互。

### 核心理念
> **"交易即游戏，决策即朝拜"**

我们不提供枯燥的交易工具，而是创造一个充满仪式感的交易世界。优秀的加密产品应该寓教于乐，将社交、游戏和财富增长融为一体，而非单纯追求功能堆砌。

### 市场痛点与解决方案

| 市场痛点 | EZ Trading解决方案 |
|---------|------------------|
| 交易体验枯燥乏味，缺乏参与感 | 科技风格全息界面，每个操作都有多维反馈 |
| 决策过程复杂，信息过载 | AI神谕一键解读，将分析转化为明确建议 |
| 用户留存困难，缺乏社区粘性 | 游戏化成就系统与多级裂变邀请机制 |
| 增长受限，缺乏社交传播动力 | 朋友邀请传送门，多级奖励激励分享 |
| 交易学习曲线陡峭 | 一键跟单机制，无需专业知识即可参与 |

## 🏗️ 项目架构

EZ Trading由六大核心模块组成：

### 1. 科技神谕终端
- 全息风格界面，充满未来科技感的交互体验
- 每个操作都有视觉、声音和微振动反馈
- 神秘的解析过程，增强决策的仪式感

### 2. AI解币实验室
- 投入EAT能量启动AI深度分析
- 全维度币种解析，输出直观评分和预测
- 高科技可视化分析结果，重点标记关键信号

### 3. 多级裂变系统
- 科技感传送门邀请好友机制
- 三级邀请奖励，构建个人影响力网络
- 邀请排行榜和专属特权解锁

### 4. 游戏化激励中心
- 每日朝拜AI神像获取EAT能量
- 连续签到累积奖励，提高用户留存
- 成就系统和等级提升路径

### 5. 一键跟单系统
- 科技执行舱直观设置交易参数
- EAT能量驱动自动交易策略
- 实时执行状态可视化监控

### 6. EAT能量池
- 多层级能量池质押系统
- EAT代币价值由NAV机制托底
- 不接管用户资金，降低信任门槛

## 💻 技术栈

### 前端
- **框架**: Vue 3
- **构建工具**: Vite
- **动效库**: GSAP/Three.js
- **风格**: 科技风格UI组件
- **响应式设计**: 移动优先原则

### 后端
- **运行环境**: Node.js
- **API框架**: Express
- **数据库**: MongoDB
- **身份验证**: JWT
- **类型安全**: TypeScript

### 游戏化系统
- **激励引擎**: 自研游戏化激励系统
- **社交网络**: 多级裂变算法
- **动态平衡**: 经济模型自平衡机制

### 多智能体系统
- **工作流引擎**: n8n
- **AI集成**: Claude、Perplexity、ChatGPT等API
- **数据分析**: Python数据科学栈

### 区块链技术
- **主链**: Solana
- **合约框架**: Anchor
- **代币标准**: SPL Token
- **扩展支持**: SUI和TON(计划中)
